25개 로스쿨 LEET 환산식 총정리

서울대·고려대·연세대를 비롯한 25개 법학전문대학원의 2026학년도 LEET 환산공식, 반영비율, 영어 P/F 기준을 한 페이지에 정리했습니다. 학교별 환산 방식이 모두 달라서 같은 표준점수라도 학교에 따라 유불리가 크게 갈립니다.

2026학년도 모집요강 기준 · 최종 업데이트: 2026-05-03

이 가이드를 읽기 전에

LEET 반영비율은 1단계 전형에서 LEET 점수가 차지하는 비중입니다. 같은 비율이라도 학교마다 환산 방식(표점합 / 백분위 / 환산표)이 달라서 실제 유불리는 다르게 나옵니다.

예를 들어 한양대(50%)와 경희대(50.9%)는 비율은 비슷하지만, 한양대는 만점 기준 비율 환산식이고 경희대는 표점합에 계수를 곱하는 선형식이라 LEET 고득점자에게 더 유리한 학교가 다릅니다.

본 가이드의 "내 점수로 직접 계산" 버튼을 누르면 사이트 메인의 학교별 환산점수 탭으로 이동해 모든 학교의 환산점수를 한 번에 비교할 수 있습니다.

서울대

국내 유일 백분위 기반 환산 · 정성평가 80점 별도
LEET 반영비율30.0%
총점 구성LEET 60 + 학부 60 + 정성평가 80 = 200점
영어P/F (TEPS 387 / TOEFL 107 이상)

LEET 환산

서울대는 표준점수가 아닌 백분위를 사용합니다. 언어 백분위와 추리 백분위를 4:6으로 가중평균한 뒤 0.6을 곱해 60점 만점으로 환산합니다.

LEET 환산점수 = (언어백분위 × 0.4 + 추리백분위 × 0.6) × 0.6

학부 환산

학점 백분위에 0.6을 곱해 60점 만점으로 환산합니다.

⚠️ 정량점수(LEET + 학부 = 120점)는 1단계 정량 평가만 결정합니다. 최종 합격은 정성평가 80점이 더해진 200점 만점에서 결정되며, 정성평가 점수는 공개되지 않습니다.

LEET 환산점수 = 언어 환산점수(별첨2-A) + 추리 환산점수(별첨2-B)

LEET 환산

표점합 = 언어 표준점수 + 추리 표준점수 → 환산표1 룩업

환산표는 비선형이라 130점 이상 구간에서는 1점 차이가 0.5점, 100점 이하 구간에서는 1점 차이가 2~3점까지 벌어집니다.

LEET 환산

해당 연도 만점자의 표준점수를 분모로 한 비율을 사용합니다. 2026학년도 기준 언어 만점 73.1점, 추리 만점 97.6점.

A = 언어표점 / 73.1
B = 추리표점 / 97.6
C = (A + B) / 2
C ≥ 0.95 → 40점
0.45 ≤ C < 0.95 → 12 + (C − 0.45) × 56
C < 0.45 → 12점

최상위권에게는 만점이 쉽게 나오지만 중상위권은 변별이 큰 식입니다.

학부 100점 환산점수가 60 이상이면 6 + (x − 60) × 0.35, 60 미만이면 기본점수 6점.
